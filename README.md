# LockCLI

- A CLI to lock the terminal while being afk.
- Usage: `[lock, unlock]`

# LockCLI from Source

- Have JDK 8+ Installed
- Compile main class and run in any IDE

# LockCLI from JAR

- Have JDK 8+ Installed
- Run java `-jar LockCLI.jar`
- Make sure to be in the same directory as the JAR to run it
